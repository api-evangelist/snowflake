The Observe CLI will enable both users and agents to operate on observability data and context through a growing set of reusable skills — structured workflows for common tasks like investigating incidents, tracing failures or validating changes. Some of these workflows will be able to run autonomously in the background, while others will be invoked interactively. The model will support both agents handling routine tasks and engineers directly executing and guiding more complex investigations, potentially from environments like Claude Code.
